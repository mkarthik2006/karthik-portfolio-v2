# Karthik Portfolio V2

Modern portfolio built with React + Vite, focused on backend engineering projects, microservices architecture, and production-ready implementation stories.

## Tech Stack

- React 18
- Vite 5
- React Router
- Lucide React

## Deployment

This project is Netlify-compatible and configured for static SPA hosting.

- Build command: `npm run build`
- Publish directory: `dist`
- SPA redirect: `public/_redirects` with `/* /index.html 200`
